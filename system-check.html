Priority Fix List

1. Open Play

ยังเป็น Legacy Stable Mode / loader. ควรย้ายโค้ด Open Play ตัวจริงกลับมาเป็นไฟล์ openplay.html โดยตรงในรอบถัดไป เพื่อไม่ต้องพึ่ง external raw fallback.

2. Supabase Security

Tournament tables เปิด public write สำหรับ prototype. ก่อนใช้หลายสนามจริง ต้องเพิ่ม Organizer/Admin permission.

3. Data Consistency

Open Play ใช้ psh_* tables, Tournament ใช้ psh_tournament_* tables, Team Battle ยังใช้ localStorage. ต้องค่อยรวม data model ในอนาคต.

4. Mobile Test

ต้องทดสอบบนมือถือจริง: iPhone Safari, Android Chrome, LINE browser, Facebook in-app browser.
