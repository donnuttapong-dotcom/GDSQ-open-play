ลำดับใช้งานจริง

1
Setup Event
สร้าง/เลือก event, สนาม, วันเวลา, จำนวนคอร์ท
Open Event Setup
2
Player Check-in
ส่ง QR/link ให้ผู้เล่นลงชื่อและเลือก level
Open Check-in
3
Queue Control
Organizer ตรวจ Ready / Rest / Left ก่อนจับคู่
Open Queue
4
Court Board
เลือก Auto Generate หรือ Manual Pair ในหน้าเดียว
Open Court Board
5
Start / Cancel / Confirm
Start match, Cancel แบบเก็บ score draft, หรือ Confirm score เพื่อบวกสถิติ
Open Match Control
6
Summary & Ranking
หลังจบงาน copy message ส่ง ranking + match history เข้ากลุ่ม
Open Summary

ใช้มือถือ 2–3 เครื่อง: เครื่องหนึ่ง Check-in, เครื่องหนึ่ง Organizer, เครื่องหนึ่ง Summary
Check-in ผู้เล่นอย่างน้อย 8 คน
ทดสอบ Auto Generate 2 คอร์ท
ทดสอบ Manual Pair โดยเลือกคนซ้ำ — ต้องเลือกไม่ได้
Start match แล้วผู้เล่นต้องหายจาก dropdown
Cancel match แล้วผู้เล่นต้องกลับ Ready และ score draft ยังอยู่
Confirm score แล้ว ranking ต้องเปลี่ยน
